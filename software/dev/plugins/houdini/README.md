# Houdini Integration

This package will contain Houdini-specific pipeline integrations:

- Shelf tools for asset loading and publishing
- HDAs (Houdini Digital Assets) for pipeline operations
- Python panels for pipeline UI inside Houdini
- USD/Solaris workflows (future)

## Development

Add `pipeline/` to HOUDINI_PATH or houdini.env so Houdini can
find the lightwarp package.
